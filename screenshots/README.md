# Screenshots Directory

This directory should contain the following screenshots of the application:

## Required Screenshots:

1. **home-landing.png** - Home page showing the landing section with navigation
2. **product-collections-sweets.png** - Sweets and Chocolate collections view
3. **register-page.png** - Registration form
4. **email-verification.png** - Email verification screen after registration
5. **login-page.png** - Login page
6. **dashboard-user.png** - User dashboard after login
7. **dashboard-admin.png** - Admin dashboard with product management

## How to Capture Screenshots:

1. Run the development server: `npm run dev`
2. Navigate to each page/route
3. Take screenshots using:
   - Windows: Win + Shift + S
   - Mac: Cmd + Shift + 4
   - Linux: PrtScn or Shift + PrtScn
4. Save the images with the names listed above in this directory
5. Update the README.md if needed

## Tips:
- Use full browser window for consistency
- Capture at 1920x1080 resolution if possible
- Show the application in action (with real data)
- Ensure the browser dev tools are closed for clean screenshots
